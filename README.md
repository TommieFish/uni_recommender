## Introduction

To run this code: 
```bash
npm install
```
Navigate to C://.../uni_recommender
Run: 
```bash
npm install @auth0/nextjs-auth0
npm install @supabase/supabase-js
```
And all the dependencies will be installed! Then, run it locally...
